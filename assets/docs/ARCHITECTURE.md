## API Response Handling

### Success Response (2xx)
```javascript
// Frontend receives:
{
  status: 200,
  data: {
    // Actual resource data
  }
}

// Handled by:
// 1. authenticatedFetch() parses JSON
// 2. Returns data to caller
// 3. Component updates UI
```

### Error Response (4xx/5xx)
```javascript
// Backend returns:
{
  status: 401,
  detail: "Unauthorized"
}

// Handled by:
// 1. authenticatedFetch() checks status
// 2. If 401: Tries to refresh token
// 3. If refresh fails: Redirects to login
// 4. Otherwise: Throws error to caller
```
